## 부가효과 태그 전체 목록 (빈도 내림차순)

> 새 태그를 추가할 때는 이 목록에서 가장 유사한 기존 태그를 우선 사용한다.

### 전투 판정 계열
- `무적판정` — 공격 받지 않는 무적 프레임 (가장 많이 쓰임, 411회)
- `패링` — 특정 공격을 막는 패링 판정
- `가드판정` — 가드로 처리되는 판정
- `경직저항증가` — 피격 시 경직 저항
- `반격` — 특정 조건에서 반격 발동

### 피해 증가 계열
- `피해증가` — 일반적인 피해 증가
- `파티원피해증가` — 파티 전체 피해 증가
- `궁극기피해증가` — 궁극기 피해 증가
- `강화특수스킬피해증가` — 강화 특수 스킬 피해 증가
- `속성이상피해증가` — 속성 이상 피해 증가
- `관입 피해증가` — 관입(방어무시) 피해 증가
- `이상피해증가` — 이상 상태 피해 증가
- `받는피해감소` — 받는 피해 감소
- `피해감소` — 피해 감소
- `데미지버프` — 범용 데미지 버프
- `피해량버프` — 피해량 버프

### 그로기/스턴 계열
- `높은그로기배율` — 그로기 수치 누적 배율 높음
- `그로기수치증가` — 그로기 수치 증가 효과
- `그로기배율증가` — 그로기 배율 증가
- `그로기약체화배율증가` — 그로기 상태 약체화 배율 증가
- `그로기` — 그로기 상태 유발

### 스탯 버프 계열
- `공격력증가` — 공격력 버프 (자신)
- `파티공격력증가` — 파티 공격력 버프
- `치명타확률증가` — 치명타 확률 증가
- `치명타피해증가` — 치명타 피해 증가
- `관통력 증가` — 관통력(방어 무시율) 증가
- `관통률증가` — 관통률 증가
- `이상축적효율증가` — 속성 이상 축적 효율 증가 (자신)
- `파티원이상축적효율증가` — 파티원 전체 속성 이상 축적 효율 증가
- `이상마스터리증가` — 이상 마스터리 증가
- `충격력증가` — 충격력 증가
- `차지속도증가` — 차지 속도 증가
- `이상 축적 효율 상승` — 이상 축적 효율 상승 (`이상축적효율증가`로 통일 권장)
- `이상 축적 효율 증가` — 이상 축적 효율 증가 (`이상축적효율증가`로 통일 권장)

### 자원 회복 계열
- `HP회복` — HP 회복 (자신 기준; `체력회복`과 통일)
- `파티원 회복` — 파티원 HP 회복 (파티 대상일 때)
- `에너지회복` — 에너지 자원 회복
- `기운회복` — 기운 자원 회복
- `데시벨회복증가` — 데시벨(궁극기 게이지) 회복량 증가
- `꿈세계 게이지 회복` — 꿈세계 게이지 회복
- `HP소모` — HP를 소모하는 스킬

## 적 디버프 태그 전체 목록

적 디버프 태그는 에이전트가 **타깃에게 직접 부착하거나 발생시키는 상태·효과**다.
파티원이 발동하는 효과와 연동되는 반응 메커니즘은 적 디버프가 아닌 시너지로 분류한다.

| 태그 | 설명 |
|------|------|
| `빙염` | 미야비 전용 — 적에게 부착, 서릿발 발동 시 서리열파괴로 전환 |
| `서리열` | 미야비 전용 — 서리열파괴 후 적이 서리열 상태에 진입 |
| `서리열파괴` | 미야비 전용 — 빙염 제거 후 서리 피해 발동 |
| `정복` | 정복 디버프 부여 |
| `정복 비례 피해증가` | 정복 스택 비례 피해 증가 |
| `감전` | 감전 상태 이상 |
| `한기` | 한기 상태 |
| `혼돈` | 혼돈 상태 (에이전트가 직접 발동하는 경우) |
| `극성 혼돈` | 극성 혼돈 상태 부여 |
| `속성축적저항감소` | 속성 축적 저항 감소 |
| `얼음 속성 피해 저항감소` | 얼음 속성 피해 저항 감소 |
| `적 받는피해 증가` | 적이 받는 피해 증가 |

## 시너지 태그 전체 목록 (45종)

시너지는 파티 구성이나 특정 조건에서 발동하는 협동 효과다.

### 속성 시너지 (5종)
- `속성시너지-물리`, `속성시너지-불`, `속성시너지-얼음`, `속성시너지-전기`, `속성시너지-에테르`

### 진영/클래스 시너지
- `진영시너지` — 같은 진영 파티 시너지 (가장 많이 쓰임)
- `클래스 시너지-격파`, `클래스 시너지-이상`, `클래스 시너지-지원`, `클래스 시너지-방어`, `클래스 시너지-명파`

- `심안`, `깨우침`, `절명`

### 에이전트 전용 시너지
- `서리와 달 자세` — 미야비 전용 (내리는 서리 소모로 발동하는 차지 자세)

- `서리와 달 자세` — 미야비 전용

- `발화점` — 버니스 전용

- `분개` — 반악 전용

## 대분류별 전형적인 태그 패턴

> 스킬 종류에 따라 자주 붙는 태그 조합 참고용

| 대분류 | 전형적인 부가효과 | 자주 없는 효과 |
|--------|-----------------|---------------|
| 회피스킬 | `무적판정` | 공격력 버프류 |
| 지원스킬 (패링) | `무적판정`, `패링`, `높은그로기배율` | HP 회복류 |
| 지원스킬 (빠른지원) | `무적판정` | — |
| 궁극기/콤보공격 | `무적판정` + 특수 효과 | — |
| 핵심스킬 (추가능력) | 피해/버프류 (무적판정 없음) | 무적판정 |
| 핵심스킬 (핵심패시브) | 피해증가, 스탯 버프 | 무적판정 |
| 일반공격 | 없거나 `경직저항증가` | 무적판정 |
